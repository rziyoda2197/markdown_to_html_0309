# 📄 Markdown to HTML Converter CLI

Simple CLI tool that converts Markdown text into basic HTML.

## 🚀 Features

- Converts headings:
  - `#` → `<h1>`
  - `##` → `<h2>`
  - `###` → `<h3>`
- Converts normal lines to `<p>` tags
- Reads markdown file and generates HTML output

## ▶️ Run

```bash
python main.py
